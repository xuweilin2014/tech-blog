## 🧪 写作说明

仓库中的文章大致分为几类：

- **原理梳理**：尽量从机制和结构上理解一个主题
- **源码分析**：围绕具体框架或组件的实现展开
- **实验记录**：偏验证、偏动手过程
- **专题笔记**：围绕某个具体问题进行集中整理

有些文章会写得比较系统，有些则更偏向某个具体问题的研究。  
部分内容会随着理解加深持续补充和修订。

---

## 👀 适合怎样阅读

你可以按自己的兴趣选择阅读方式：

- **按目录读**：从感兴趣的主题直接进入
- **按专题读**：沿着某一条主线连续看，例如 Netty / Redis / OSPF
- **按推荐阅读读**：从上面的入口文章开始
- **按系列顺序读**：对于标有“一 / 二 / 三”的文章，建议按顺序阅读
